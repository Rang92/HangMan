# HangMan
Rang Salih - 10690972

HangMan - Try to guess the word of given length, letter by letter, but remember you have limited turns.

Note: Too easy? Try with evil-mode on.

#### Features

* Use on-screen keyboard
* Change settings
  * Length word
  * Turns allowed
  * Evil-mode ON/OFF
* See highscores
  * On homescreen
  * On winner screen
* Restart game
  * On homescreen
  * On loser screen
* Game instructions
* Give username
* Username in highscores list
